# DevHelper - Simple CLI Tool

A beginner-friendly C project with 3 useful features.

## Files (Only 4 files!)

```
final/
├── main.c           # Main program with menu
├── weather.c        # Weather checker
├── project_setup.c  # Check dev environments
└── git_helper.c     # Git commands helper
```

## How to Compile

Just one command:

```bash
gcc main.c weather.c project_setup.c git_helper.c -o devhelper
```

## How to Run

```bash
./devhelper
```

## Features

1. **Weather Checker** - Get real weather for any city
2. **Project Setup Helper** - Check if Python, Node.js, Java, Git are installed
3. **Git Helper** - Easy git commands (status, add, commit, push, pull, log)

## What You'll Learn

- Multiple C files in one project
- Function declarations
- String handling (fgets, sscanf, sprintf)
- File I/O (fopen, fgets, fclose)
- System commands (system())
- Switch-case statements
- Basic JSON parsing

## Requirements

- GCC compiler
- curl (for weather)
- git (for git helper)
